## 📖 使用说明

1. 输入创作主题
2. 选择诗词风格（现代诗/古风诗/俳句）
3. 选择情感倾向
4. 点击"生成诗词"按钮
5. 等待AI创作完成
6. 可以选择复制文本或导出图片分享
