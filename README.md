# AI诗词创作助手

一个优雅的AI驱动的诗词创作工具，支持多种AI模型，可以生成现代诗、古风诗和俳句等多种风格的诗词作品。

## ✨ 特性

- 🤖 支持多个AI模型（Deepseek、Moonshot、通义千问）
- 📝 支持多种诗词风格（现代诗、古风诗、俳句）
- 🎨 丰富的主题选项（浅色、深色、水墨风）
- 📏 灵活的排版方式（横排、竖排）
- 🎭 多样的情感倾向
- 💾 本地历史记录
- 📤 支持导出图片
- 📋 一键复制分享

## 🛠️ 技术栈

- 纯原生JavaScript实现
- CSS变量实现主题切换
- LocalStorage实现数据持久化
- HTML5 Canvas实现图片导出

## 🚀 快速开始

1. 克隆项目到本地：

2. 在浏览器中打开`index.html`即可使用

## 📖 使用说明

1. 输入创作主题
2. 选择诗词风格（现代诗/古风诗/俳句）
3. 选择情感倾向
4. 点击"生成诗词"按钮
5. 等待AI创作完成
6. 可以选择复制文本或导出图片分享
